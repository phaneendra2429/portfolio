Coffee Shop Sales

Data Analysis Dashboard

Project Objectives:

The main objective of this project was to create an end-to-end Excel dashboard for data analysis with practical applications. The project aimed to demonstrate the process of customizing Excel charts, adding borders, changing properties, and enhancing visual appeal to make data analysis more effective. Additionally, the project focused on uploading files and screenshots to Git repositories for collaboration and sharing project insights.

GitHub Link

Project Details

Steps Performed:
  • Customized Excel charts by adding borders and changing properties like line width and colors.
  • Uploaded files and screenshots to Git repositories for collaboration and sharing insights.
  • Edited and added PDF files to document data analysis findings and solutions.
  • Analyzed data and created a dashboard to visualize project insights effectively.
Outcomes:
  • Successfully customized Excel charts, enhancing visual appeal and data analysis effectiveness.
  • Efficiently uploaded files to Git repositories for collaboration on project insights.
  • Documented data analysis findings and insights in PDF files for better project documentation and analysis.
